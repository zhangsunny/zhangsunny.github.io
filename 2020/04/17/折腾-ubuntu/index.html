安装注意事项

windows下安装双系统需要注意：

  • 关闭windows电源选项中的快速启动
  • 关闭bios的快速启动
  • bios中security boot 选择开启并选择 other os
  • 注意UEFI安装系统，需要三个分区/（主分区ext4，100-200G，重装时需要格式化，软件会装到这里，需要足够大），EFI（逻辑分区，100M），/home（逻辑分区ext4，重装系统时不用格式化，存储个人文件的位置），设置EFI所在分区为引导位置（双系统也要设置从ubuntu的引导启动）

对于windows系统安装时不要直接使用一键安装，使用winNT安装

修复grub

https://blog.csdn.net/weixin_43491112/article/details/84064142

在安装ubuntu系统时，有时会出现无法添加grub的问题，此时的系统其实已经安装好了，只是没有添加grub引导。可以重启后，进入启动盘的试用ubuntu，通过控制台手动修复grub

  • sudo add-apt-repository ppa:yannubuntu/boot-repair && sudo apt-get update
  • sudo apt-get install -y boot-repair && boot-repair
  • sudo update-grub

开机自动挂载硬盘

https://blog.csdn.net/qq_35451572/article/details/79541106

  • 查看分区选择需要的分区编号：sudo fdisk -l
  • 查看所选分区对应的UUID：sudo blkid /dev/sda3
  • 创建挂载目录：sudo mkdir /mnt/Data (mnt文件夹是默认挂载，图像界面系统默认挂载在media)
  • 修改配置文件：sudo gedit /etc/fstab
  • 在文件末尾添加： 其中，第一个数字：0表示开机不检查磁盘，1表示开机检查磁盘； 第二个数字：0表示交换分区，1代表启动分区（Linux），2表示普通分区；挂载的分区是在WIn系统下创建的分区，磁盘格式为ntfs

微软雅黑字体

https://blog.csdn.net/qq183837971/article/details/78235144

可以从windows系统下的C:\Windows\Fonts文件夹下复制需要的字体文件，注意必须是ttf格式

  • 下载 msyh.ttf 和 msyhbd.ttf (注意是ttf文件，不是ttc文件)
  • 复制到/usr/share/fonts/winFonts文件中，文件夹自建
  • sudo chmod 644 /usr/share/fonts/winFonts/*.ttf
  • sudo mkfontscale （创建雅黑字体的fonts.scale文件，它用来控制字体旋转缩放）
  • sudo mkfontdir （创建雅黑字体的fonts.dir文件，它用来控制字体粗斜体产生）
  • sudo fc-cache -fv （建立字体缓存信息，也就是让系统认识雅黑）
  • 安装unity-tweak-tool，设置字体

配置环境

安装N卡驱动

传统方式

  • sudo add-apt-repository ppa:graphics-drivers/ppa
  • sudo apt-get update
  • ubuntu-drivers devices (查看所需驱动版本)
  • sudo apt-get install nvidia-driver-***(版本号)

若提示无法找到add-apt-repository命令，则执行sudo apt-get install software-properties-common

若提示fetch error，则执行 sudo apt-get update

简单方式

https://blog.csdn.net/tjuyanming/article/details/80862290

  • sudo apt-get update
  • sudo ubuntu-drivers autoinstall

安装驱动完成后 sudo reboot

可选方式

  • 从 software&updates 的 additional drivers 中选择驱动进行安装
