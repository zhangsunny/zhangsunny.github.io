安装配置git bash

  • 绑定用户
  • 生成SSH密钥，中间设置全按回车使用默认选项
  • 找到用户文件夹下的.ssh文件夹中的id_rsa.pub密钥，将内容全部复制。打开GitHub_Settings_keys 页面，新建new SSH Key，将pub的内容粘贴到Key中。测试是否配置成功：

安装配置hexo

  • 在GitHub网站新建仓库repository，命名方式必须是：[username].github.io
  • 首先需要安装node.js和npm（包管理工具），下载地址：https://nodejs.org/en/download/
  • 配置国内镜像源，或者使用代理
  • 初始化博客文件夹
  • hexo常用命令（需要进入到博客文件夹下）
    • 新建md文档：hexo new “title” 或 hexo n “title”
    • 生成静态网页：hexo generate 或 hexo g
    • 启动本地服务器：hexo server 或 hexo s （默认地址：localhost:4000）
    • 部署到GitHub：hexo deploy 或 hexo d （需要先配置GitHub）

部署到GitHub

  • 在项目根文件夹下的_config.yml文件的末尾修改deploy配置：
  • 安装部署工具（–save选项是将模块安装到项目目录下，并在package文件的dependencies节点写入依赖）
  • hexo部署命令 hexo d，中间会弹出GitHub登陆窗口
  • hexo通常执行流程：hexo g -> hexo d
